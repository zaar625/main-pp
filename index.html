Intro

프론트앤드 개발자를 꿈꾸는 저를 소개합니다.

안녕하세요.
하고자 하는 일에 있어서 끊임없이 도전하고 부딛히며 경험으로 성장하는 이상윤입니다.

세련됨과 멋진 이펙트를 가진 웹사이트에 매료되어 프론트앤드라는 직업군을 알게 되었고
"비슷한 사이트를 만들기"를 작은 목표로 강의, 책, 문서 등 찾아보며 웹에 대한 지식을 채워나갔습니다.
이 과정에서 React, Vue, typescript 라는 프레임워크와 언어를 알게 되었고, 왜? 라는 질문과 함께 공부하며 기량을 향상 시켰습니다.

아직 많이 부족하지만 빠르게 변화하는 시대에 맞춰 "why"라는 질문을 바탕으로 지속적인 공부와 자기계발을 통해 더 멋진 내일이 올 수 있도록 발전해 나가고, 성장하기위해 끊임없이 노력하는 프론트엔드 개발자가 되겠습니다.
감사합니다.

01. React

[소개]

단순히 멋진 UI만드는 것이 다인줄 알았던 생각을 벗어나게 해준 저의 첫번째 작품입니다.
The Movie Database Api 로부터 Axios를 이용하여 영화 데이터를 가져오는 방법부터 react-router-dom을 사용하여 페이지를 만드는 과정까지 배우며 만들었습니다.

View site

02. Vue

[소개]

대부분의 사람들이 React가 좋다라는 말만으로 하나만 배우고 싶지 않았습니다. 동일한 작품으로 Vue를 이용하여 무엇이 좋고, 불편한지 알고 프레임워크를 선택하고 싶은 마음에 만든 작품입니다.

View site
03. react site

쇼핑몰을 돌아다니며 이젠 옷만 보는 것이 아닌 이 기능 어떻게 만들지? 라는 궁금증까지 생기게 되어 만들게 된 작품입니다.
로그인 구현부터 제품 필터기능 그리고 redux-toolkit을 이용하여 상태관리까지 구현하였습니다.
또한 firebase를 통해 게시물등록까지 제작하였습니다.
해당 프로젝트는 지속적으로 매장 위치 Map, typescript 전환 등 배우고 적용하고, 기능들을 추가 할 예정입니다.

View site
04. standard

웹 표준을 준수하여 만든 작품입니다. HTML5, CSS3의 개념들을 알게 되었고, skip메뉴, IR효과 등을 통해 웹접근성을 높여 사용자가 이용에 불편이 없도록 하였습니다.

View site
05. responsive

미디어 쿼리를 이용하여 디바이스 종류에 따라 유연하게 보여지는 반응형으로 제작하였습니다. scss의 편리함 그리고 모듈화의 중요성을 느끼게 되었고, "scrollreveal" 라이브러리를 이용하여 좀 더 편리하게 자연스러운 이펙트효과를 주었습니다.
